## 🎯 Features เพิ่มเติมสำหรับระดับประถม-มัธยม

### 1. ระบบผู้ปกครอง (Parent System)
**ตาราง Database**:
```sql
parent_students
├── id (PK)
├── parent_id (FK → users)
├── student_id (FK → users)
├── relationship (father/mother/guardian)
└── created_at
```

**ฟีเจอร์**:
- ผู้ปกครองเชื่อมโยงกับนักเรียน
- ดูความคืบหน้าการเรียน
- ดูรายงานคะแนน
- รับการแจ้งเตือน
- ชำระเงินแทนบุตรหลาน

---

### 2. ระบบแบบทดสอบ (Quiz System)
**ตาราง Database**:
```sql
quizzes
├── id (PK)
├── course_id (FK → courses)
├── session_id (FK → course_sessions)
├── title
├── description
├── time_limit (minutes)
└── created_at

quiz_questions
├── id (PK)
├── quiz_id (FK → quizzes)
├── question_text
├── question_type (multiple_choice/true_false/short_answer)
├── options (JSON)
├── correct_answer
└── points

quiz_attempts
├── id (PK)
├── quiz_id (FK → quizzes)
├── student_id (FK → users)
├── score
├── total_points
├── started_at
├── completed_at
└── answers (JSON)
```

**ฟีเจอร์**:
- แบบทดสอบหลังเรียน
- แบบทดสอบกลางภาค/ปลายภาค
- ดูผลคะแนน
- ดูเฉลย (หลังทำเสร็จ)
- ระบบคะแนนสะสม

---

### 3. ระบบการบ้าน (Assignment System)
**ตาราง Database**:
```sql
assignments
├── id (PK)
├── course_id (FK → courses)
├── session_id (FK → course_sessions)
├── tutor_id (FK → tutors)
├── title
├── description
├── due_date
├── max_points
└── created_at

assignment_submissions
├── id (PK)
├── assignment_id (FK → assignments)
├── student_id (FK → users)
├── file_url
├── submitted_at
├── score
├── feedback
└── graded_at
```

**ฟีเจอร์**:
- อาจารย์มอบหมายการบ้าน
- นักเรียนส่งการบ้าน (อัปโหลดไฟล์)
- ตรวจและให้คะแนน
- แจ้งเตือนก่อน deadline
- ดูประวัติการส่ง

---

### 4. ระบบประกาศ (Announcement System)
**ตาราง Database**:
```sql
announcements
├── id (PK)
├── branch_id (FK → branches, nullable)
├── course_id (FK → courses, nullable)
├── created_by (FK → users)
├── title
├── content
├── priority (low/normal/high/urgent)
├── target_audience (all/students/parents/tutors)
└── created_at
```

**ฟีเจอร์**:
- ประกาศจากอาจารย์
- ประกาศจาก Admin
- ประกาศเฉพาะสาขา
- ประกาศเฉพาะคอร์ส
- แจ้งเตือนผ่าน SSE

---

### 5. ระบบส่งเอกสาร (Material Delivery System)
**ตาราง Database**:
```sql
materials
├── id (PK)
├── course_id (FK → courses)
├── session_id (FK → course_sessions)
├── title
├── description
├── file_url
└── created_at

material_deliveries
├── id (PK)
├── material_id (FK → materials)
├── enrollment_id (FK → enrollments)
├── delivery_method (kerry_express/digital)
├── tracking_number
├── status (pending/shipped/delivered)
├── shipped_at
└── delivered_at
```

**ฟีเจอร์**:
- ส่งเอกสารผ่าน Kerry Express
- ติดตามสถานะการส่ง
- ประวัติการส่ง

## 📊 Analytics & Reporting

### Metrics ที่ต้องติดตาม

#### 1. นักเรียน
- จำนวนนักเรียนทั้งหมด
- จำนวนนักเรียนใหม่ต่อเดือน
- อัตราการคงอยู่ (Retention Rate)
- อัตราการเข้าเรียน

#### 2. คอร์ส
- จำนวนคอร์สที่เปิดสอน
- คอร์สยอดนิยม
- อัตราการลงทะเบียน
- อัตราการเรียนจบ

#### 3. การเรียน
- ชั่วโมงเรียนทั้งหมด
- อัตราการเข้าเรียน
- ความคืบหน้าเฉลี่ย
- คะแนนเฉลี่ย

#### 4. การเงิน
- รายได้ต่อเดือน
- รายได้ต่อสาขา
- รายได้ต่อคอร์ส
- อัตราการชำระเงินสำเร็จ
